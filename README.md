# SpringReactShop

#### 介绍
一个简单的、简陋的在线商城、管理系统

- 后端采用springboot+mybatis
- 后台管理使用react+antd
- 商城使用react+antd-mobile

[springboot后端查看具体介绍](./shop-spring/README.md)

[admin管理查看具体介绍](./shop-admin-react/README.md)
