# SpringShop

#### 介绍
reactShop的后端，使用springboot2框架

前后端分离，react全家桶+springboot+jwt

[配套前端ReactShop](https://gitee.com/zhuyunlong2018/ReactShop)

#### 本机环境

1. JDK: v1.8.0_221
2. maven: v3.6.0
3. redis: v3.2.1
4. mysql: v5.5.53

#### 框架依赖

